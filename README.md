# 맑은사주 (Malgeun Saju)

계산은 정직하게, 해석은 겁 없이 — 투명한 만세력 기반 무료 사주 웹.

## 차별점 (시장 공백 3가지를 정면으로)

1. **계산 vs 해석 분리 표시** — 파란 배지(100% 재현 가능한 계산) / 금색 배지(전통 통설 요약)를 구분. 시장에 전무한 기능.
2. **쉬운말 / 전문가 모드 토글** — 같은 결과를 두 언어로. 역시 시장 전무.
3. **계산 투명성** — 진태양시(경도+균시차), 역사적 표준시(1954–61 UTC+8:30), 서머타임(1948–60, 1987–88), 절기 천문 계산(태양 황경), 경계 근접 경고까지 전 과정 공개.
4. 광고 없음 · 겁주기 없음 · 결제 유도 팝업 없음.

## 검증

`/tmp` 테스트 하네스로 4개 실명식(수동 검증 완료된 사주)과 대조 — 사주 4주·대운수·대운 방향·십성·십이운성·신살·세운 이벤트 전부 일치.

## 구조

- `js/manseryeok.js` — 계산 엔진 (절기 이분탐색, 일주 앵커 산술, 진태양시, 대운)
- `js/data.js` — 해석 텍스트 (easy/expert 이중 모드, 출처 라벨)
- `js/app.js` — UI, 공유 URL, 모드 토글
- 정적 사이트 — 빌드 불필요, GitHub Pages 배포
